# Brand assets

Drop the DrMax logo here as either:

- `logo.webp` — preferred (smaller file, transparent background supported)
- `logo.png` — fallback if you don't have WebP

The game tries WebP first, then PNG. If neither exists, the HUD top bar shows the fallback text "DrMax Pharmacy" instead.

## Sizing

Whatever resolution you have works — the game scales the logo to fit the HUD height (~36 px tall), preserving aspect ratio. For crispness on retina screens, **300–500 px tall** in the source file is plenty.

## Recommended

- **Format:** WebP, lossless or quality ≥ 90
- **Background:** transparent (the HUD is dark navy)
- **Padding:** keep tight crop — the HUD has its own dark background, no need for the white pill around the logo unless that's part of the brand mark
